# fivem-Web-shop
## วิธีใช้งาน 
* Clone / Download Repo นี้
* ลง Sql
* เปลี่ยน database name ในไฟล์ system/sql.php
* เปลี่ยน steam api ในไฟล์ config.php

## ฟังก์ชัน 
* Login ด้วย Steam
* ระบบ Item ในตัว
* ระบบข้อมูลส่วนตัว (ชื่อ เงิน ...)
* เติมเงินผ่าน coupon 
* ระบบซื้อของในเกม
* ระบบ admin (เพิ่ม coupon , เพิ่มสินค้าในเว็บ) เขียนเพิ่มเอง (ขี้เกียจ55)
